# Architecture Review (Tier 3)

## Domain Scope
Periodic assessment of the co-worker's own architecture. Should sub-agents be promoted or consolidated? Are there coverage gaps? Is the skill library evolving appropriately? Should new agent perspectives be added?

## Review Questions
- Which agent perspectives are most/least used?
- Are any sub-agents redundant?
- What user requests fall outside current agent coverage?
- Are skills capturing repeatable workflows effectively?
- Is the memory/ schema still adequate or does it need new sections?
