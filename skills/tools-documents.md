# Tool Reference: Document Storage

## When Available
Look for Google Drive, Box, or Notion document tools in the current session. Use them to search, read, upload, and list documents.

## Memory Mapping
- Proposal documents → referenced in memory/role/portfolio/pipeline.md
- Grant agreements → referenced in memory/role/grantees/[name]/profile.md
- Board reports → referenced in memory/role/briefings/
- Standards documents → referenced in memory/role/standards/

## Without External Access
- Documents are referenced by description in memory files
- User shares document content directly in conversation
- Co-worker extracts and stores key information in relevant memory files

## Connected Skills
- proposal-eval.md reads proposal documents when available
- portfolio-review.md accesses board report templates
- meeting-prep.md pulls relevant documents for meeting context
