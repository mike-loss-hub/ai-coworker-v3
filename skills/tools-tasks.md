# Tool Reference: Task Management

## When Available
Look for Asana, Linear, or Notion task management tools in the current session. Use them to create, update, list, and complete tasks.

## Memory Mapping
- External tasks sync to memory/role/tasks/active.md
- Completed external tasks sync to memory/role/tasks/completed.md
- Blocked tasks with external blockers sync to memory/role/tasks/blocked.md

## Without External Access
- All task tracking in memory/role/tasks/ files (this is the default)
- Co-worker manages tasks entirely through local markdown files
- User can assign, update, and complete tasks through conversation

## Connected Skills
- task-delegation.md creates tasks in both local and external systems
- briefing.md reads tasks from local files (or external if available)
- meeting-debrief.md creates action items as tasks
