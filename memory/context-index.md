# Context Index — AI Co-Worker Memory Map

**Read this file first on every session.**

This memory system has three layers:
- **human/** — Portable. Travels with the user across roles and jobs. Identity, working style, feedback history, generalizable skills, growth log.
- **role/** — Role-bound. Specific to this position. Portfolio, contacts, grantees, standards, ecosystem. Stays behind when you leave.
- **bridge/** — Transitional. Generalizable lessons, decision patterns, relationship archetypes, anti-patterns. Helps onboard to a new role.
