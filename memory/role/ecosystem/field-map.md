## Agent Coordination Protocol Landscape (Updated 2026-05-02)

**Protocol convergence:** MCP (Anthropic, Nov 2024) and A2A (Google, Apr 2025) are now both under the Linux Foundation's Agentic AI Foundation (AAIF), co-founded Dec 2025 by OpenAI, Anthropic, Google, Microsoft, AWS, Block. MCP = agent-to-tool, A2A = agent-to-agent. A2A v1.0 stable Q1 2026, MCP v2.0 Q1 2026, joint interoperability spec draft Q2 2026. Fragmentation risk is LOW.

**Gap:** No one is building the education-specific coordination layer on top of A2A+MCP — how agents share learner context, verify credentials between agents, enforce privacy constraints, and coordinate in local-first environments.

**Key projects:**
- **Google ADK** — hierarchical agent trees with native A2A support. Open source. General purpose, not education-specific.
- **OAP (Open Agent Protocol)** — open-source agent interoperability atop LangGraph/LangChain. Promising protocol-level work, not education-specific.
- **OpenClaw** — open-source multi-agent orchestration for parallel execution. Runtime, not protocol spec.
- **Microsoft Agentic Framework** — open source Jan 2026. Enterprise multi-agent, not education.
- **VoltAgent** — curating 2026 AI agent research papers (memory, evaluation, workflows). Research resource.

**Education-specific agent work:** Research growing (multi-agent debate, affective scaffolding, teacher co-orchestration) but focused on pedagogy, not coordination protocols. No open protocol spec for education agent coordination found.
