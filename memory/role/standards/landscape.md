# Standards Landscape

**Last Updated:** 2026-04-28

## 1EdTech Comprehensive Learner Record (CLR) 3.0
- **Status:** Draft specification, comment period open (closes May 20, 2026)
- **Relevance:** Primary standard for structured learner records in education. Directly relevant to Longitudinal Memory pillar.
- **Key Contact:** Linda Zhang (VP Standards)
- **Portfolio Alignment:** LearnChain building CLR-compatible LRS. TrustLayer bridging CLR to VC ecosystem.
- **Watch:** CLR 3.0 data model changes could affect LearnChain integration timeline.

## W3C Verifiable Credentials (VC) 2.0
- **Status:** Candidate Recommendation (nearing final)
- **Relevance:** Broader credential verification standard. Essential for Provenance pillar.
- **Key Contact:** David Kowalski (CG Co-Chair)
- **Portfolio Alignment:** CredBridge using VC format for provenance metadata. TrustLayer bridging VC to CLR.
- **Watch:** VC 2.0 finalization timeline could affect CredBridge and TrustLayer deliverables.

## W3C Decentralized Identifiers (DIDs) 1.0
- **Status:** W3C Recommendation (finalized)
- **Relevance:** Identity layer for credential holders. Relevant to Provenance and Local-First pillars.
- **Portfolio Alignment:** VerifyNet proposal (in pipeline) uses DIDs for decentralized verification.

## T3 Innovation Network / CTDL
- **Status:** Active development, version 2.5
- **Relevance:** Credential Transparency Description Language — connects education and workforce credential descriptions.
- **Key Contact:** Diana Foster (T3 Director)
- **Portfolio Alignment:** Potential workforce pilot in pipeline. CTDL alignment needed for workforce credential portability.

## Model Context Protocol (MCP) / Agent-to-Agent (A2A)
- **Status:** Converging under Linux Foundation AAIF (Dec 2025). A2A v1.0 stable Q1 2026, MCP v2.0 Q1 2026. Joint interoperability spec draft Q2 2026.
- **Governance:** AAIF co-founded by OpenAI, Anthropic, Google, Microsoft, AWS, Block.
- **Consensus:** MCP = agent-to-tool (vertical), A2A = agent-to-agent (horizontal). Complementary, not competing.
- **Relevance:** Core transport layer for Agent Orchestration pillar. Fragmentation risk now LOW.
- **Portfolio Alignment:** PathWeaver using MCP for multi-agent demo. Protocol stability improving. Education-specific coordination layer still missing — no one defining how education agents share learner context over A2A+MCP.
- **Watch:** Q2 2026 joint interoperability spec draft. 78% of enterprise AI teams report MCP-backed agents in production (Apr 2026).
- **Updated:** 2026-05-02

## Open Badges 3.0
- **Status:** 1EdTech Final Release
- **Relevance:** Micro-credential representation. Complementary to CLR for granular achievements.
- **Portfolio Alignment:** Indirect — grantees may need to ingest Open Badges data.
