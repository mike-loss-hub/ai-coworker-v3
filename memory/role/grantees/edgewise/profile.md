## Project Summary
EdgeWise is building an edge inference engine that enables learner context models to run on low-resource devices (Chromebook-class hardware). This is critical for local-first architecture — learners in low-connectivity settings need AI capabilities without constant cloud dependence. Initial benchmarks on Chromebook hardware showed higher latency than projected. Team is investigating model quantization and distillation approaches.

## Key Deliverables
1. Edge inference benchmark baseline (Mar 2026 — ⚠️ Below target)
2. Optimized inference engine v1.0 (May 18, 2026)
