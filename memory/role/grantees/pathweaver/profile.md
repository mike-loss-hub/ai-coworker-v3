## Project Summary
PathWeaver is building the reference implementation for multi-agent coordination around shared learner records. Three AI agents (advising, tutoring, career coaching) will demonstrate coordinated operation around a single learner profile using MCP as the transport layer. The Year 2 demo is the portfolio's flagship deliverable — showing that agent orchestration with shared context is possible, safe, and beneficial.

## Key Deliverables
1. Agent coordination protocol specification (Dec 2025 — ✅ Complete)
2. Single-agent MCP integration (Feb 2026 — ✅ Complete)
3. Multi-agent demo with 3 agents (June 1, 2026)
4. Evaluation framework for agent coordination quality (June 2026)
