## Project Summary
OpenSync is building a CRDT-based sync protocol specifically designed for credential-grade data — data that requires stronger consistency guarantees than typical collaborative editing. The protocol handles offline operation, conflict resolution for structured credentials, and selective sync (learners choose what to share with which institution).

## Key Deliverables
1. Sync protocol spec v0.2 (Mar 2026 — ✅ Complete)
2. Sync protocol spec v0.3 with conflict resolution (May 20, 2026)
3. Reference implementation (Sep 2026)
4. Interop testing with LearnChain LRS (Nov 2026)
